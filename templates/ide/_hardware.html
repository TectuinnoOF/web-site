Microcontrolador Tectuinno

Hardware diseñado específicamente para trabajar con nuestro IDE. Potente, versátil y fácil de usar para todos tus proyectos

Arquitectura Híbrida

Combina un ESP-12F (Wi-Fi) con una FPGA Tang Nano 9K para una lógica programable flexible.

Memoria Integrada

El ESP-12F cuenta con RAM interna para ejecución y memoria Flash para firmware y configuración.

Visualización y Control

Incluye 8 LEDs auxiliares y 2 bloques de switches DIP de 4 bits para interactuar con tus proyectos.

Conectividad Avanzada

Módulos integrados SIMCOM A7670E (4G) y E50-TTL-500 (RF) para comunicaciones robustas.

Conectores para Pantalla

Dispone de conectores específicos para pantallas P10 monocolor, facilitando proyectos de visualización.

Alimentación Estabilizada

Regulador de voltaje AMS1117-3.3V con capacitores de desacoplo para una alimentación limpia y estable.

Pines de Expansión

Pines adicionales disponibles para conectar periféricos externos a través de SPI, UART o GPIO desde la FPGA.
